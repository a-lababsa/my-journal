## Le site

Ce journal est généré avec [Hugo](https://gohugo.io/), un générateur de sites statiques. Il est hébergé sur [GitHub Pages](https://pages.github.com/) — aucun serveur, aucune base de données, aucun cookie de traçage.

Le thème *Journal Intime* est une création originale, conçue pour ce projet.

## Typographie

- **Corps du texte** — Merriweather ou Lora, pour l'œil qui lit lentement
- **Titres** — Caveat, une écriture à la main
- **Code** — JetBrains Mono

Les polices sont chargées depuis [Google Fonts](https://fonts.google.com/).

## Accessibilité

Le site propose un mode dyslexie (bouton *Aa*) et un mode sombre (bouton *☽*), activables à tout moment. Il est conçu pour être lisible, pas spectaculaire.
